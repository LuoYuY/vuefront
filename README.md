## 启动项目
# 后端
``` 
来自我的项目 blog 启动SpringBooot
```

# 数据库
```
见Mysql.txt
```
# 安装启动
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
